Linux

Currently I run (GNU/)Linux on all my computers, although my main desktop also dualboots with Windows which I find myself using more for gaming so I won't be discussing it any further. I run all of my server applications on a Dell Thin Client for low power consumption, both this machine and my laptop run Debian as my choice of distro.

Apologies in advance for the wall of text, I realize that some images would probably be a good idea to make this a more attention-grabbing read.

I will start with my laptop since it is where I've learned the most.
I am using Debian with DWM as a window manager. Although this is quite a strange combination, I like that Debian has a lot of information online so I can solve any problems I have, while DWM gives me something to tinker with when I am bored. I actually really enjoy using a Tiling WM. For a few years before I started using Linux I was using AutoHotKey to make my Windows experiance more streamlined, so when I found out that there were so many programs out there on Linux that used Vim-like binding, I quickly found myself switching to those. With that said I really like qutebrowser as a web browser, and do my file browsing with nnn (I should eventually get around to linking these). The keyboard-focused workflow I have on my laptop has definitely taught me a lot along the way setting it up, and although some people would argue that they are impractical for daily usage, this is incorrect becuase trackpads are much less conveinient or precise than keyboard shortcuts, even if they take a little more time initially to setup and learn.
